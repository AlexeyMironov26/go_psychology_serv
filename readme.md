Это телеграм бот для Службы Психологической Помощи мтуси - @spp_mtuci_bot.
Прежде чем запустить бинарный/исходный файл с исполняемым кодом бота вам нужно установить в оболочке, из который вы будете запускать его (.\botmain.exe или ./botmain (linux) или go run botmain.go, если запускаете файл с исходным кодом и есть компилятор для го) переменную окружения с токеном бота как: 
export TELEGRAM_BOT_TOKEN="токен этого бота"
или для windows (cmd)
setx TELEGRAM_BOT_TOKEN "токен этого бота" (токен является вещью, с помощью которой можно получить полный дуступ и контроль над ботом, его важно хранить в секрете и безопасности)
(после назначения новой переменной окружения нужно перезапустить сеанс в оболочке и только после этого запускать код бота)
