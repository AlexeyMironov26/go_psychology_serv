Это телеграм бот для Службы Психологической Помощи мтуси - @spp_mtuci_bot.
Написан на Python 3.12.6. (желательна либо такая же версия либо выше)
Чтобы запустить бот нужно перейти в директорию с проектом и выполнить команды:
1. python -m venv venv (создание виртуального окружения)
2. .\venv\Scripts\activate (активация виртуального окружения, если оно находится в текущей директории - для Windows)
3. pip install -r requirements.txt (установка в активированное виртуальное окружение зависимостей проекта)
После этого уже можно запускать сам тг бот (ИЗ АКТИВИРОВАННОГО venv, из папки с файллом spp_bot.py) командой: 
1. python spp_bot.py
Прекратить работу можно, если нажать Ctrl+C в терминале, в котором был запущен бот.
